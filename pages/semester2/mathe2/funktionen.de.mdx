# Funktionen

## Totale Funktion $$f: D \rightarrow C$$

Eine totale Funktion ist eine Regel, die jedes Element aus einer Menge
**𝐷** (der Definitionsmenge) genau einem Element aus einer anderen Menge
**𝐶** (der Wertemenge) zuordnet.

Einfacher gesagt: Für jedes $x$ aus **𝐷** gibt es genau ein $y$ ($f(x)$) aus **𝐶**.

z.B.

- $f(1)=2$
