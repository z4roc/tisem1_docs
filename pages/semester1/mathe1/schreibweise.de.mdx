# Schreibweise der Mathematik

Die Schreibweise der Mathematik kann auf den ersten Blick sehr verwirrend wirken.
Hier ist eine Übersicht über einige der wichtigsten Symbole und Konventionen (auf Deutsch), die in der Mathematik verwendet werden:

- $:$ beudetet "ist" oder "gilt"

- $=$ bedeutet "ist gleich"

- $\forall$ bedeutet "für alle"

- $\in$ bedeutet "ist Element von, ist in"

- $\notin$ bedeutet "ist genau ein Element von"

- $\mathbb{R}$ bedeutet die Menge der reellen Zahlen

- $\mathbb{N}$ bedeutet die Menge der natürlichen Zahlen

- $\mathbb{Z}$ bedeutet die Menge der ganzen Zahlen

- $\mathbb{Q}$ bedeutet die Menge der rationalen Zahlen

- $\mathbb{C}$ bedeutet die Menge der komplexen Zahlen

- $\exists$ bedeutet "es existiert, oder auch es gibt mindestens 1"

- $\exists!$ bedeutet "es existiert genau 1"

- $\nexists$ bedeutet "es existiert nicht, oder auch es gibt kein"

- $\neg$ bedeutet "nicht"

- $\Rightarrow$ bedeutet "daraus folgt"

- $\Leftrightarrow$ bedeutet "genau dann, wenn"

- $:= $ bedeutet "ist definiert als, wird definiert als, wird festgelegt als"

$:= $ kann man auch mit der Zuweisung einer Variable in der Programmierung vergleichen.

Bsp: $\forall x \in \mathbb{R} : x^2 \geq 0$

Bedeutet: Für alle $x$ aus den reellen Zahlen gilt: $x^2$ ist größer oder gleich 0.
