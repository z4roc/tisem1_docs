# Python Syntax

Die Syntax von Python ist sehr einfach und übersichtlich.
Statt geschweiften Klammern wie in Java oder C++ werden Einrückungen verwendet.

Am ende einer Zeile wird kein Semikolon benötigt.

Variablen müssen nicht deklariert werden, sondern werden automatisch erstellt, wenn sie zum ersten Mal verwendet werden.

```python
# Basic Python Code

# Print "Hello, World!"
print("Hello, World!")

# Define a variable
x = 5

# Check if x is greater than 10
if x > 10:
    print("x is greater than 10")
else:
    print("x is not greater than 10")
```
