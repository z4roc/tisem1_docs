# Programmieren 1

Im Modul Programmieren 1 lernst du die Grundlagen der Programmierung
in der Progammiersprache Python

## Software

Für das Modul benötigst du folgende Software:

- [Python](https://www.python.org/downloads/)
- [Anaconda](https://www.anaconda.com/products/individual)

Für die Arbeit in PyCharm und VSCode kannst du Anaconda auslassen.

Als Code Editor stehen dir folgende zur Verfügung:

- [PyCharm](https://www.jetbrains.com/pycharm/download/)

- [Jupyter Notebook](https://jupyter.org/install)

- [Visual Studio Code](https://code.visualstudio.com/)
